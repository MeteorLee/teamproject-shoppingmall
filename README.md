# 에비네츄럴 쇼핑몰

## 개요

- 팀 프로젝트
- 2023.4.6 ~ 2023.5.22

### 기술

- Java
- Spring
- MySql

## 맡은 역할

쇼핑몰의 결제 구현을 담당함

### 카카오페이
![팀프로젝트카카오 drawio](https://github.com/user-attachments/assets/6ecdae36-7d03-432b-923c-e733c4f1cebb)


- 외부 문서를 통해 결제 기능 구현
- 백엔드 서버에서 카카오 서버로 결제 승인 요청을 보냄
- 결제 기능의 성격에 맞게 결제 상태가 변할 때마다 DB에 결제 상태를 반영함
